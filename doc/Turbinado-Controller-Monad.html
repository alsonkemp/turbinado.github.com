The Controller Monad
type Controller = StateT Environment IOSource
The Controller monad is a state wrapper around the IO monad.
runController :: Controller () -> Environment -> IO EnvironmentSource
Runs a Controller computation in a particular environment. Since Controller wraps the IO monad, the result of running it will be an IO computation.
