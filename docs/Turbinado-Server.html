startServer :: Flag -> IO ()Source
Starts the server, builds the basic Environment, builds the WorkerPool, starts listening on the specified port. As soon as a request is noticed, it's handed off to a WorkerThread to be handled. Lather, Rinse, Repeat.

data WorkerPool Source

Worker Pool stuff

The WorkerPool holds each idle or busy WorkerThread. When all WorkerThreads are busy, more are created by getWorkerThread and added to the WorkerPool.
